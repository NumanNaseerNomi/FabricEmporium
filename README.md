# 👕 FabricEmporium - Online Clothing Store

A modern, full-featured e-commerce platform for clothing built with Laravel and Bootstrap. FabricEmporium provides a seamless shopping experience for fashion enthusiasts with powerful product management, secure checkout, and an intuitive admin panel.

## ✨ Features

### 👤 Customer Features
- **Product Browsing**: Browse clothing items by categories, collections, and trends
- **Advanced Filtering**: Filter by size, color, price range, brand, and style
- **Product Search**: Search for specific items with autocomplete suggestions
- **Product Details**: View high-quality images, sizes, colors, materials, and care instructions
- **Size Guide**: Interactive size chart for accurate fitting
- **Wishlist**: Save favorite items for later purchase
- **Shopping Cart**: Add/remove items, manage quantities, and save for later
- **User Accounts**: Register, login, and profile management
- **Order Tracking**: Track order status from placement to delivery
- **Order History**: View past orders and reorder easily
- **Product Reviews**: Rate and review purchased items with images
- **Secure Checkout**: Multiple payment options and address management

### 👨‍💼 Admin Features
- **Dashboard**: Analytics dashboard with sales, orders, and revenue metrics
- **Product Management**: CRUD operations for products with image upload
- **Inventory Management**: Track stock levels, sizes, and colors
- **Category Management**: Organize products by categories and subcategories
- **Order Management**: View, update, and process customer orders
- **User Management**: Manage customers and staff accounts
- **Discount & Coupon Management**: Create and manage promotions
- **Review Moderation**: Approve or reject product reviews
- **Report Generation**: Sales reports, inventory reports, and analytics
- **Site Settings**: Configure store settings and preferences
